# 🧾 Namma Santhe Ledger – Mobile App

A modern and user-friendly ledger management mobile application developed for vegetable and fruit market businesses. The app helps vendors manage customer transactions, track income & expenses, and send WhatsApp payment reminders digitally.

---

# 🚀 Features

✅ 📒 Ledger Management  
✅ 💰 Income & Expense Tracking  
✅ 📊 Analytics Dashboard  
✅ 🥦 Item Type Selection (Vegetables / Fruits / Mixed)  
✅ 🏪 Customer Type Selection (Retail / Wholesale / Hotel)  
✅ 🔍 Search & Filter Transactions  
✅ ✏️ Edit Transactions  
✅ 🗑️ Delete Transactions  
✅ 💬 WhatsApp Payment Reminder  
✅ 🌙 Dark Mode Support  
✅ 💾 Offline Local Storage  
✅ 📱 Clean & Responsive UI
